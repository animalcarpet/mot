Grid system

See an example of the grid system in use

DVSA MOT modernisation uses the grid system from bootstrap v3.0. The container is made up of 12 columns, each with a padding of 15px left and right, the container itself also has 15px padding left and right.

This table details how the container columns react depending on the size of the device/browser window it is being viewed in.
